About This App


This app was written by Chris Ruggieri for UAB IS 633 Mobile Applications Course. This app allows you to pull Customer Names and Order Histories for a ficticous company. You can select "All Customers" to pull a table of all the stores. You can select the store and pull its history or you can input the Customer ID to pull its history.
